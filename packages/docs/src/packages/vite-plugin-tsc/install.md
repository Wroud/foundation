# Installation

<Badges name="@wroud/vite-plugin-tsc" />

Install with your package manager:

::: code-group

```sh [npm]
npm install @wroud/vite-plugin-tsc
```

```sh [yarn]
yarn add @wroud/vite-plugin-tsc
```

```sh [pnpm]
pnpm add @wroud/vite-plugin-tsc
```

```sh [bun]
bun add @wroud/vite-plugin-tsc
```

:::
