# Installation

<Badges name="@wroud/navigation" />

Install with your package manager of choice:

::: code-group

```sh [npm]
npm install @wroud/navigation
```

```sh [yarn]
yarn add @wroud/navigation
```

```sh [pnpm]
pnpm add @wroud/navigation
```

```sh [bun]
bun add @wroud/navigation
```

:::
