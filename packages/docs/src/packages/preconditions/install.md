# Installation

<Badges name="@wroud/preconditions" />

Install with npm, or use your favorite package manager:

::: code-group

```sh [npm]
npm install @wroud/preconditions
```

```sh [yarn]
yarn add @wroud/preconditions
```

```sh [pnpm]
pnpm add @wroud/preconditions
```

```sh [bun]
bun add @wroud/preconditions
```

:::
